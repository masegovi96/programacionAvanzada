Actividad de Diagnóstico: Botón de Accesibilidad Web

Objetivo

Evaluar tus habilidades en desarrollo web implementando funciones de accesibilidad configurables en una página con contenido estructurado. Puedes utilizar CSS, JavaScript, HTML, PHP, etc, lo que consideres mejor para realizar dicha herramienta.

Instrucciones

Crea una página web que contenga al menos:

  • Un menú de navegación.
  • Un encabezado principal (por ejemplo, título con imagen o logo).
  • Un bloque de texto con al menos dos párrafos.

Agrega un botón fijo en la esquina superior derecha con el texto “Accesibilidad”.

Al hacer clic en el botón, se debe mostrar un panel o menú de opciones donde el usuario pueda seleccionar qué herramientas de accesibilidad desea activar o desactivar, entre al menos las siguientes (elige 4 o más):

  • Aumentar tamaño del texto.
  • Cambiar a modo de alto contraste o modo oscuro.
  • Usar una fuente legible (como OpenDyslexic o Arial).
  • Subrayar todos los enlaces.
  • Activar enfoque visible (resaltar elementos al navegar con teclado).
  • Modo lectura (ocultar elementos visuales no esenciales).

El usuario debe poder activar/desactivar cada herramienta por separado, sin necesidad de activar todas a la vez.

Usa HTML semántico, CSS y JavaScript (puedes usar frameworks si los justificas). Se evaluará tanto la funcionalidad como la organización del código.

Entrega: Archivo HTML completo (con CSS y JS embebidos o enlazados) o enlace a un repositorio.

Tiempo estimado: 60 minutos.
