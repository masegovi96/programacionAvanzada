Counting Sort

¿Qué es el Counting Sort?

Counting Sort (Ordenamiento por conteo) es un algoritmo de ordenamiento no comparativo que funciona contando la frecuencia de cada elemento en un rango específico y luego utilizando esa información para construir el arreglo ordenado. Es eficiente para datos con un rango limitado de valores.

¿Cómo funciona Counting Sort?

  1. Determinar el rango de los valores en el arreglo.
  2. Construir un arreglo de conteo para almacenar la frecuencia de cada valor.
  3. Modificar el arreglo de conteo para que cada posición contenga la suma acumulativa de las frecuencias.
  4. Construir el arreglo ordenado utilizando el arreglo de conteo.

Pasos del Algoritmo (Detallado)

  1. Encontrar el valor máximo en el arreglo.
  2. Inicializar un arreglo de conteo con ceros.
  3. Contar la frecuencia de cada elemento en el arreglo.
  4. Calcular la suma acumulativa en el arreglo de conteo.
  5. Construir el arreglo ordenado utilizando el arreglo de conteo.

Complejidad

Caso Complejidad
Peor caso O(n + k)
Mejor caso O(n + k)
Promedio O(n + k)

donde n es el número de elementos en el arreglo y k es el rango de los valores.

Ventajas

  • Muy eficiente para datos con un rango limitado de valores.
  • No realiza comparaciones entre elementos.
  • Es estable si se implementa correctamente.

Desventajas

  • No es adecuado para datos con un rango muy amplio.
  • Requiere memoria adicional para el arreglo de conteo.
  • No es un algoritmo in-place.
Animación: Cómo funciona el Counting Sort
Ejemplo en PHP

function countingSort($arr) {
    $max = max($arr);
    $count = array_fill(0, $max + 1, 0);

    foreach ($arr as $num) {
        $count[$num]++;
    }

    $sorted = [];
    for ($i = 0; $i <= $max; $i++) {
        while ($count[$i] > 0) {
            $sorted[] = $i;
            $count[$i]--;
        }
    }

    return $sorted;
}
Ejemplo en JavaScript

function countingSort(arr) {
    const max = Math.max(...arr);
    const count = new Array(max + 1).fill(0);

    for (const num of arr) {
        count[num]++;
    }

    const sorted = [];
    for (let i = 0; i <= max; i++) {
        while (count[i] > 0) {
            sorted.push(i);
            count[i]--;
        }
    }

    return sorted;
}
Ejemplo en Python

def counting_sort(arr):
    max_val = max(arr)
    count = [0] * (max_val + 1)

    for num in arr:
        count[num] += 1

    sorted_arr = []
    for i, freq in enumerate(count):
        sorted_arr.extend([i] * freq)

    return sorted_arr
