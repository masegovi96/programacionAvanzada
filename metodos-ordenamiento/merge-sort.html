Merge Sort

¿Qué es el Merge Sort?

Merge Sort (Ordenamiento por mezcla) es un algoritmo de ordenamiento eficiente basado en el paradigma de dividir y conquistar. Divide la lista en mitades, las ordena recursivamente y luego las combina en una lista ordenada.

¿Cómo funciona Merge Sort?

  1. Dividir la lista en dos mitades.
  2. Ordenar recursivamente cada mitad.
  3. Combinar las dos mitades ordenadas en una sola lista ordenada.

Pasos del Algoritmo (Detallado)

  1. Dividir la lista en mitades hasta que cada sublista tenga un solo elemento.
  2. Combinar las sublistas ordenadas comparando los elementos de cada una.
  3. Repetir el proceso hasta que todas las sublistas estén combinadas en una lista ordenada.

Complejidad

Caso Comparaciones Divisiones Complejidad
Peor caso O(n log n) O(log n) O(n log n)
Mejor caso O(n log n) O(log n) O(n log n)
Promedio O(n log n) O(log n) O(n log n)
  • Espacio adicional: O(n) (requiere espacio adicional para las sublistas).
  • Estable: Sí (mantiene el orden relativo de elementos iguales).

Ventajas

  • Muy eficiente para listas grandes.
  • Estable y predecible en su rendimiento.

Desventajas

  • Requiere espacio adicional proporcional al tamaño de la lista.
  • Más complejo de implementar que otros algoritmos simples como Bubble Sort o Selection Sort.

Ejemplo en PHP

function mergeSort($arr) {
    if (count($arr) <= 1) {
        return $arr;
    }
    $mid = floor(count($arr) / 2);
    $left = mergeSort(array_slice($arr, 0, $mid));
    $right = mergeSort(array_slice($arr, $mid));
    return merge($left, $right);
}

function merge($left, $right) {
    $result = [];
    while (count($left) > 0 && count($right) > 0) {
        if ($left[0] <= $right[0]) {
            $result[] = array_shift($left);
        } else {
            $result[] = array_shift($right);
        }
    }
    return array_merge($result, $left, $right);
}
Ejemplo en JavaScript

function mergeSort(arr) {
    if (arr.length <= 1) {
        return arr;
    }
    const mid = Math.floor(arr.length / 2);
    const left = mergeSort(arr.slice(0, mid));
    const right = mergeSort(arr.slice(mid));
    return merge(left, right);
}

function merge(left, right) {
    const result = [];
    while (left.length && right.length) {
        if (left[0] <= right[0]) {
            result.push(left.shift());
        } else {
            result.push(right.shift());
        }
    }
    return result.concat(left, right);
}
Ejemplo en Python

def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    while left and right:
        if left[0] <= right[0]:
            result.append(left.pop(0))
        else:
            result.append(right.pop(0))
    return result + left + right
