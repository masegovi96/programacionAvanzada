Reglas de la clase

Asistencia y puntualidad

  • El pase de lista se realiza a las 7:10 AM.
  • Llegar entre las 7:11 AM y 7:15 AM se registra como retardo.
  • Después de las 7:15 AM se contabilizará como falta y no se permitirá el acceso al aula.

Uniforme

  • Es obligatorio asistir con el uniforme institucional completo y en buen estado.
  • El incumplimiento implica la pérdida del porcentaje correspondiente a este criterio.

Uso de tecnología

  • Inteligencia Artificial: La IA es una herramienta de apoyo al aprendizaje, pero no debe sustituir la capacidad de razonamiento del alumno. Queda completamente prohibido utilizarla para realizar tareas o actividades de participación de forma íntegra. El uso excesivo resultará en la anulación de la entrega con calificación de 0.
  • Integridad en exámenes: Durante los exámenes, el uso de cualquier IA está estrictamente prohibido. La detección resultará en la anulación del examen con calificación de 0.
  • Dispositivos móviles en exámenes: Durante los exámenes se retirarán los dispositivos móviles del área de trabajo. Queda completamente prohibido su uso durante la evaluación.
  • GitHub y Classroom: Es obligatorio tener una cuenta de GitHub y un repositorio correctamente configurado. Todas las entregas se gestionarán a través de GitHub y Classroom.
  • Estructura del repositorio: Deberás crear un repositorio en GitHub con el siguiente formato de nombre: Apellido_Nombre_PA_SextoCuatrimestre. En este repositorio se subirán todas las tareas, participaciones y ejercicios de exámenes. Deberás compartirlo vía Classroom en un apartado que se publicará posteriormente.

Entregas y evaluación

  • No se aceptarán entregas extemporáneas bajo ninguna circunstancia.
  • Se revisarán los logs de Git para validar la hora exacta de cada entrega.
  • Posterior a cada entrega, el alumno deberá explicar su trabajo y responder preguntas relacionadas.
  • Presta mucha atención durante las clases. Todo lo que se explica o muestra será pedido en el proyecto final y evaluado en los exámenes.

Conducta

  • No se permiten faltas de respeto en el salón. El respeto entre alumnos–alumnos, alumnos–maestro y maestro–alumnos deberá mantenerse en todo momento.
  • No tengas miedo de preguntar. No te quedes con ninguna duda. Recuerda que no hay preguntas tontas, sino tontos que no preguntan.
El docente se reserva el derecho de actualizar estas reglas a lo largo del cuatrimestre. Cualquier cambio será comunicado con anticipación.
