Proyecto Gestor Bibliotecario

Descripción del Proyecto

El Gestor Bibliotecario es un sistema web desarrollado en PHP que permite administrar el catálogo de una biblioteca. Incluye funcionalidades para registrar libros, gestionar préstamos y devoluciones, así como consultar la disponibilidad del acervo desde una interfaz web. Se conecta a una base de datos MySQL para persistir toda la información.

Tecnologías utilizadas

  • PHP — Lógica del servidor y procesamiento de consultas preparadas. (Más adelante se actualizará para trabajar con peticiones HTTP asíncronas)
  • MySQL — Base de datos relacional
  • Javascript (Fetch API / async) — Comunicación asíncrona con el servidor. (Más adelante se actualizará para incluir esta tecnología, al día de hoy no se usa en el proyecto).
  • HTML & CSS — Interfaz de usuario

Instrucciones de instalación y ejecución

Para ejecutar el proyecto necesitas un servidor local que soporte PHP y MySQL, como XAMPP.

  1. Descarga e instala XAMPP

    Descarga XAMPP desde apachefriends.org e instálalo en tu equipo. Asegúrate de incluir los módulos Apache y MySQL.

  2. Descarga el proyecto

    Descarga el archivo .zip del proyecto desde la tarjeta de abajo y descomprímelo.

  3. Coloca el proyecto en htdocs

    Copia la carpeta descomprimida gestor_bibliotecario/ dentro de C:\xampp\htdocs\ (en Windows) o el directorio equivalente de tu sistema.

  4. Importa la base de datos

    Abre phpMyAdmin en http://localhost/phpmyadmin , crea una nueva base de datos llamada gestor_bibliotecario e importa el archivo .sql que se encuentra dentro de la carpeta del proyecto.

  5. Configura la conexión a la base de datos

    Abre el archivo de configuración (normalmente config.php o conexion.php) y verifica que los datos de conexión coincidan con tu entorno local (host, usuario, contraseña y nombre de la BD).

  6. Inicia los servicios de XAMPP

    Abre el Panel de Control de XAMPP y arranca los módulos Apache y MySQL.

  7. Accede al sistema

    Abre tu navegador y navega a http://localhost/gestor_bibliotecario/ para comenzar a usar el sistema.

Notas importantes

  • El proyecto fue desarrollado para ejecutarse en un entorno local con XAMPP; no está configurado para producción.
  • Asegúrate de que el puerto 80 (Apache) y el puerto 3306 (MySQL) estén disponibles en tu máquina.
  • Si usas un sistema operativo distinto a Windows, la ruta de htdocs puede variar:
    • macOS (XAMPP): /Applications/XAMPP/htdocs/
    • Linux (LAMP): /var/www/html/
