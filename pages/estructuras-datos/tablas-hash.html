Tablas Hash

¿Qué son las tablas hash?

Una tabla hash es una estructura de datos que utiliza una función hash para mapear claves a valores. Es eficiente para operaciones como búsqueda, inserción y eliminación.

Operaciones Básicas

  • put(clave, valor) → inserta un valor asociado a una clave
  • get(clave) → busca el valor
  • remove(clave) → elimina el valor
  • containsKey(clave) → verifica si existe la clave
  • size() → retorna el tamaño de la tabla hash

Manejo de Colisiones

Una colisión ocurre cuando dos claves distintas producen el mismo índice al aplicar la función hash. Las estrategias más comunes para resolverlo son:

  • Encadenamiento (Chaining): Cada celda de la tabla contiene una lista enlazada con todos los elementos que colisionan en ese índice.
  • Direccionamiento abierto (Open Addressing): Cuando hay colisión, se busca la siguiente celda libre. Hay tres variantes:
    • Sondeo lineal: Se avanza de una en una hasta encontrar un hueco.
    • Sondeo cuadrático: Se avanza en potencias de 2 (1, 4, 9…).
    • Doble hashing: Se aplica una segunda función hash para calcular el salto.

Complejidad

Operación Caso Promedio Peor Caso
put() / insertarO(1)O(n)
get() / buscarO(1)O(n)
remove() / eliminarO(1)O(n)
Espacio totalO(n)

El peor caso O(n) ocurre cuando todas las claves colisionan en el mismo índice.

Implementaciones Comunes

  • Bases de datos (índices hash para búsqueda rápida)
  • Contadores de frecuencia de palabras o eventos
  • Cachés (memoria rápida, como Redis o Memcached)
  • Verificación de duplicados en grandes conjuntos de datos
  • Tablas de símbolos en compiladores

Ejemplo en PHP

class TablaHash {
    private $tabla = [];

    public function insertar($clave, $valor) {
        $indice = $this->hash($clave);
        $this->tabla[$indice] = $valor;
    }

    public function obtener($clave) {
        $indice = $this->hash($clave);
        return $this->tabla[$indice] ?? null;
    }

    private function hash($clave) {
        return crc32($clave) % 10; // Función hash simple
    }
}

// Ejemplo de uso
$tabla = new TablaHash();
$tabla->insertar("clave1", "valor1");
$tabla->insertar("clave2", "valor2");
echo $tabla->obtener("clave1"); // Imprime "valor1"
Ejemplo en JavaScript

// Clase TablaHash
class TablaHash {
    constructor() {
        this.tabla = new Array(10);
    }

    hash(clave) {
        let hash = 0;
        for (let i = 0; i < clave.length; i++) {
            hash += clave.charCodeAt(i);
        }
        return hash % 10;
    }

    insertar(clave, valor) {
        const indice = this.hash(clave);
        this.tabla[indice] = valor;
    }

    obtener(clave) {
        const indice = this.hash(clave);
        return this.tabla[indice];
    }
}

// Ejemplo de uso
const tabla = new TablaHash();
tabla.insertar("clave1", "valor1");
tabla.insertar("clave2", "valor2");
console.log(tabla.obtener("clave1")); // Imprime "valor1"
Ejemplo en Python

class TablaHash:
    def __init__(self):
        self.tabla = [None] * 10

    def hash(self, clave):
        return sum(ord(c) for c in clave) % 10

    def insertar(self, clave, valor):
        indice = self.hash(clave)
        self.tabla[indice] = valor

    def obtener(self, clave):
        indice = self.hash(clave)
        return self.tabla[indice]

# Ejemplo de uso
tabla = TablaHash()
tabla.insertar("clave1", "valor1")
tabla.insertar("clave2", "valor2")
print(tabla.obtener("clave1")) # Imprime "valor1"
