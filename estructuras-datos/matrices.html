Matrices

¿Qué son?

Estructura bidimensional organizada en filas y columnas. �atil para representar datos tabulares o espaciales.

Operaciones Básicas

  • Acceder a un elemento: matriz[i][j]
  • Modificar un elemento: matriz[i][j] = nuevo_valor
  • Recorrer matriz: doble for (anidados)
  • Insertar fila/columna (en lenguajes dinámicos como Python)
  • Eliminar fila/columna

Implementaciones Comunes

  • Juegos tipo tablero
  • Imágenes digitales
  • Algoritmos numéricos

Ejemplo en PHP

                            // Crear una matriz bidimensional
                            $matriz = [
                                [1, 2, 3],
                                [4, 5, 6],
                                [7, 8, 9]
                            ];

                            // Acceder a un elemento
                            echo $matriz[1][2]; // Imprime 6

                                [7, 8, 9]
