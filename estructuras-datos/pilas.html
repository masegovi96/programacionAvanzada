Pilas

¿Qué son las pilas?

Las pilas son estructuras de datos lineales que siguen el principio LIFO (Last In, First Out), donde el último elemento en entrar es el primero en salir.

Operaciones Básicas

  • push(elemento) → inserta elemento arriba
  • pop() → elimina el último elemento
  • peek() o top() → consulta el elemento superior
  • isEmpty() → verifica si está vacía
  • size() → retorna el tamaño de la pila

Implementaciones Comunes

  • Recursión
  • Reversión de cadenas
  • Deshacer acciones

Ejemplo en PHP

// Crear una pila usando un arreglo (LIFO: último en entrar, primero en salir)
$pila = [];
array_push($pila, "A");  // Pila: [A]
array_push($pila, "B");  // Pila: [A, B]
array_push($pila, "C");  // Pila: [A, B, C]

// Consultar el elemento en el tope sin sacarlo (peek)
$tope = end($pila);      // Obtiene "C"
echo "Tope: $tope\n";    // Imprime: Tope: C

// Verificar si la pila está vacía
if (!empty($pila)) {
    echo "La pila no está vacía\n";
}

// Sacar el último elemento (pop)
$elemento = array_pop($pila); // Pila: [A, B]
echo "Sacado: $elemento\n";  // Imprime: Sacado: C
Ejemplo en JavaScript

// Crear una pila usando un arreglo (LIFO: último en entrar, primero en salir)
const pila = [];
pila.push("A");  // Pila: ["A"]
pila.push("B");  // Pila: ["A", "B"]
pila.push("C");  // Pila: ["A", "B", "C"]

// Consultar el elemento en el tope sin sacarlo (peek)
const tope = pila[pila.length - 1]; // "C"
console.log("Tope:", tope);          // Imprime: Tope: C

// Verificar si la pila está vacía
if (pila.length > 0) {
    console.log("La pila no está vacía");
}

// Sacar el último elemento (pop)
const elemento = pila.pop();        // Pila: ["A", "B"]
console.log("Sacado:", elemento);   // Imprime: Sacado: C
Ejemplo en Python

# Crear una pila usando una lista (LIFO: último en entrar, primero en salir)
pila = []
pila.append("A")  # Pila: ["A"]
pila.append("B")  # Pila: ["A", "B"]
pila.append("C")  # Pila: ["A", "B", "C"]

# Consultar el elemento en el tope sin sacarlo (peek)
tope = pila[-1]               # "C"
print("Tope:", tope)          # Imprime: Tope: C

# Verificar si la pila está vacía
if pila:
    print("La pila no está vacía")

# Sacar el último elemento (pop)
elemento = pila.pop()         # Pila: ["A", "B"]
print("Sacado:", elemento)    # Imprime: Sacado: C
