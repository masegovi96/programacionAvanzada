Listas Enlazadas

¿Qué son las listas enlazadas?

Las listas enlazadas son estructuras de datos lineales donde cada elemento (nodo) contiene un valor y una referencia al siguiente nodo en la lista.

Operaciones Básicas

  • insertAtBeginning(valor) → insertar al inicio
  • insertAtEnd(valor) → insertar al final
  • insertAt(pos, valor) → insertar en una posición específica
  • deleteAt(pos) → eliminar en una posición específica
  • search(valor) → buscar un valor en la lista
  • traverse() → recorrer elementos

Complejidad

Implementaciones Comunes

  • Base para implementar otras estructuras (pilas, colas, deques)
  • Recoleción de basura (Garbage Collection) en lenguajes administrados
  • Historial de navegación y listas de reproducción de música
  • Sistemas de archivos con asignación de bloques no contiguos
  • Representación de polinomios y listas dispersas (sparse lists)
Animación: Cómo funcionan las listas enlazadas
Ejemplo en PHP

// Clase Nodo: representa cada elemento (nodo) de la lista
class Nodo {
    public $valor;      // Dato almacenado en el nodo
    public $siguiente;  // Referencia al siguiente nodo

    public function __construct($valor) {
        $this->valor = $valor;
        $this->siguiente = null; // Al crearse, no apunta a ningún nodo
    }
}

// Crear tres nodos con valores A, B y C
$nodo1 = new Nodo("A");
$nodo2 = new Nodo("B");
$nodo3 = new Nodo("C");

// Enlazar los nodos para formar la lista: A → B → C
$nodo1->siguiente = $nodo2;
$nodo2->siguiente = $nodo3;
// $nodo3->siguiente queda en null (fin de la lista)

// Recorrer la lista nodo por nodo
$actual = $nodo1;
while ($actual != null) {
    echo $actual->valor . " ";   // Imprime: A B C
    $actual = $actual->siguiente; // Avanza al siguiente nodo
}

// Clase Nodo: representa cada elemento de la lista
class Nodo {
    constructor(valor) {
        this.valor = valor;        // Dato almacenado en el nodo
        this.siguiente = null;     // Referencia al siguiente nodo (null = fin)
    }
}

// Crear tres nodos con valores A, B y C
const nodo1 = new Nodo("A");
const nodo2 = new Nodo("B");
const nodo3 = new Nodo("C");

// Enlazar los nodos para formar la lista: A → B → C
nodo1.siguiente = nodo2;
nodo2.siguiente = nodo3;
// nodo3.siguiente queda en null (fin de la lista)

// Recorrer la lista nodo por nodo
let actual = nodo1;
while (actual !== null) {
    console.log(actual.valor);     // Imprime: A, luego B, luego C
    actual = actual.siguiente;     // Avanza al siguiente nodo
}
Ejemplo en Python

# Clase Nodo: representa cada elemento de la lista
class Nodo:
    def __init__(self, valor):
        self.valor = valor       # Dato almacenado en el nodo
        self.siguiente = None    # Referencia al siguiente nodo (None = fin)

# Crear tres nodos con valores A, B y C
nodo1 = Nodo("A")
nodo2 = Nodo("B")
nodo3 = Nodo("C")

# Enlazar los nodos para formar la lista: A -> B -> C
nodo1.siguiente = nodo2
nodo2.siguiente = nodo3
# nodo3.siguiente queda en None (fin de la lista)

# Recorrer la lista nodo por nodo
actual = nodo1
while actual is not None:
    print(actual.valor)       # Imprime: A, luego B, luego C
    actual = actual.siguiente # Avanza al siguiente nodo
