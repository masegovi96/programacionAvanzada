Colas

¿Qué son las colas?

Las colas son estructuras de datos lineales que siguen el principio FIFO (First In, First Out), donde el primer elemento en entrar es el primero en salir.

Operaciones Básicas

  • enqueue(elemento) → agregar al final
  • dequeue() → quitar del frente
  • front() → consultar el primero sin quitarlo
  • isEmpty() → verificar si la cola está vacía
  • size() → obtener el tamaño de la cola

Implementaciones Comunes

  • Planificadores de tareas
  • Simulación de procesos
  • Cola de impresión
Animación: Cómo funcionan las colas
Ejemplo en PHP

// Crear una cola usando un arreglo (FIFO: primero en entrar, primero en salir)
$cola = [];
array_push($cola, "A");   // Cola: [A]
array_push($cola, "B");   // Cola: [A, B]
array_push($cola, "C");   // Cola: [A, B, C]

// Consultar el primero sin sacarlo (front)
$primero = $cola[0];      // "A"
echo "Primero: $primero\n"; // Imprime: Primero: A

// Verificar si la cola está vacía
if (!empty($cola)) {
    echo "La cola no está vacía\n";
}

// Sacar el primer elemento (dequeue)
$elemento = array_shift($cola); // Cola: [B, C]
echo "Sacado: $elemento\n";    // Imprime: Sacado: A
Ejemplo en JavaScript

// Crear una cola usando un arreglo (FIFO: primero en entrar, primero en salir)
const cola = [];
cola.push("A");  // Cola: ["A"]
cola.push("B");  // Cola: ["A", "B"]
cola.push("C");  // Cola: ["A", "B", "C"]

// Consultar el primero sin sacarlo (front)
const primero = cola[0];           // "A"
console.log("Primero:", primero);  // Imprime: Primero: A

// Verificar si la cola está vacía
if (cola.length > 0) {
    console.log("La cola no está vacía");
}

// Sacar el primer elemento (dequeue)
const elemento = cola.shift();    // Cola: ["B", "C"]
console.log("Sacado:", elemento); // Imprime: Sacado: A
Ejemplo en Python

# deque (double-ended queue) es más eficiente que una lista para colas
from collections import deque

cola = deque()
cola.append("A")  # Cola: deque(["A"])
cola.append("B")  # Cola: deque(["A", "B"])
cola.append("C")  # Cola: deque(["A", "B", "C"])

# Consultar el primero sin sacarlo (front)
primero = cola[0]           # "A"
print("Primero:", primero)  # Imprime: Primero: A

# Verificar si la cola está vacía
if cola:
    print("La cola no está vacía")

# Sacar el primer elemento (dequeue) - O(1) con deque
elemento = cola.popleft()         # Cola: deque(["B", "C"])
print("Sacado:", elemento)        # Imprime: Sacado: A
