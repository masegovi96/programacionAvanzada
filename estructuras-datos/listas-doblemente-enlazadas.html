Listas Doblemente Enlazadas

¿Qué son las listas doblemente enlazadas?

Las listas doblemente enlazadas son estructuras de datos donde cada nodo contiene un valor, una referencia al nodo siguiente y una referencia al nodo anterior.

Operaciones Básicas

  • insertFront(valor) → insertar al inicio
  • insertBack(valor) → insertar al final
  • deleteFront() → eliminar al inicio
  • deleteBack() → eliminar al final
  • search(valor) → buscar un valor en la lista
  • traverseForward(), traverseBackward() → recorrer elementos hacia adelante y hacia atrás

Implementaciones Comunes

  • Navegadores
  • Listas musicales, historial

Ejemplo en PHP

// Clase Nodo: guarda el dato, el nodo anterior y el siguiente
class Nodo {
    public $valor;      // Dato almacenado
    public $anterior;   // Enlace al nodo anterior
    public $siguiente;  // Enlace al nodo siguiente

    public function __construct($valor) {
        $this->valor    = $valor;
        $this->anterior = null;
        $this->siguiente = null;
    }
}

// Crear tres nodos
$nodo1 = new Nodo("A");
$nodo2 = new Nodo("B");
$nodo3 = new Nodo("C");

// Enlazar: A <-> B <-> C
$nodo1->siguiente = $nodo2;
$nodo2->anterior  = $nodo1;
$nodo2->siguiente = $nodo3;
$nodo3->anterior  = $nodo2;

// Recorrer hacia adelante: imprime A B C
$actual = $nodo1;
while ($actual != null) {
    echo $actual->valor . " ";
    $actual = $actual->siguiente;
}

// Recorrer hacia atrás: imprime C B A
$actual = $nodo3;
while ($actual != null) {
    echo $actual->valor . " ";
    $actual = $actual->anterior;
}
Ejemplo en JavaScript

// Clase Nodo: guarda el dato y referencias al anterior y siguiente
class Nodo {
    constructor(valor) {
        this.valor    = valor;     // Dato almacenado
        this.anterior = null;      // Enlace al nodo anterior
        this.siguiente = null;     // Enlace al nodo siguiente
    }
}

// Crear tres nodos
const nodo1 = new Nodo("A");
const nodo2 = new Nodo("B");
const nodo3 = new Nodo("C");

// Enlazar: A <-> B <-> C
nodo1.siguiente = nodo2;
nodo2.anterior  = nodo1;
nodo2.siguiente = nodo3;
nodo3.anterior  = nodo2;

// Recorrer hacia adelante: imprime A, B, C
let actual = nodo1;
while (actual !== null) {
    console.log(actual.valor);
    actual = actual.siguiente;
}

// Recorrer hacia atrás: imprime C, B, A
actual = nodo3;
while (actual !== null) {
    console.log(actual.valor);
    actual = actual.anterior;
}

# Clase Nodo: guarda el dato y referencias al anterior y siguiente
class Nodo:
    def __init__(self, valor):
        self.valor    = valor  # Dato almacenado
        self.anterior = None   # Enlace al nodo anterior
        self.siguiente = None  # Enlace al nodo siguiente

# Crear tres nodos
nodo1 = Nodo("A")
nodo2 = Nodo("B")
nodo3 = Nodo("C")

# Enlazar: A <-> B <-> C
nodo1.siguiente = nodo2
nodo2.anterior  = nodo1
nodo2.siguiente = nodo3
nodo3.anterior  = nodo2

# Recorrer hacia adelante: imprime A, B, C
actual = nodo1
while actual is not None:
    print(actual.valor)
    actual = actual.siguiente

# Recorrer hacia atrás: imprime C, B, A
actual = nodo3
while actual is not None:
    print(actual.valor)
    actual = actual.anterior
